DATALAKE INFORMATION SYSTEM

A Data Lake Information System is a software platform or framework designed to efficiently store, organize information of the data stored in a data lake. It is a centralized location that allows you to analyse data and give knowledge of the data stored across lake. We have provided three types of assistances which will be helpful for technical team to know the properties of the tables files etc. to take decisions and for business team to check the data present in table and decide the table usage and lastly client specific module to help understand the dependent tables, path and other information.
