ABOUT

Enabling Modern Datastore to improve extract information of the data stored in ADLS

TECH ASSIST

It provides the information of the file’s tables present in the lake which is useful for the decisions of the performance cost and scalability.

BIZ ASSIST

It provides the information of the data present in the table files etc., which will be helpful for business user to understand the data and decide with the usage of the schema.

KPIF ASSIST

It is a client specific module, which helps to provide the information of the KPI’s dependents like base tables, curated tables along with the location of the KPI so on.
